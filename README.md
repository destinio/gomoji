# Welcome to Gomoji!

Gomoji is a simple emoji CLI tool to quickly find and copy emojis to your clipboard.

## Installation

```bash
./install.sh
```

```bash
make install
```

## Usage

```bash
gomoji [search term]
```

## Examples
